# Zenin

Productivity system focused on projects, consistency, and visible progress.

## Feature List

### A. Auth & Persistence

- Google OAuth authentication
- Local-first state
- Google Drive sync & backups
- Export data (JSON / CSV)
- Read-only public progress sharing

---

### B. Project System (Core)

- Create / edit / archive projects
- Pause / resume projects
- Project weighting & daily contribution
- Project-level goals & targets
- Composite (parent) projects

---

### C. Project Types & Data Inputs

- Task-based projects
- Time-based projects (manual sessions)
- Output-based projects (custom units)
- Habit projects (binary / partial)
- Knowledge / reflection projects
- Activity-integrated projects (GitHub, Strava)
- Manual override for all project types

---

### D. Task & Schedule Management

- Tasks scoped to projects
- One-off & recurring tasks
- Task difficulty / weight
- Calendar view (day / week / month)
- Drag-and-drop scheduling

---

### E. Daily Progress System

- Per-project daily progress scoring
- Global daily progress aggregation
- Progress normalization & caps
- Grace days & miss tolerance
- Activity audit log

---

### F. Streak System

- Per-project streaks
- Global consistency streak
- Grace days & streak freezes
- Streak history tracking

---

### G. Gamification & Rewards

- XP & level system
- Currency & rewards
- Unlockable themes & cosmetics
- Achievements & milestones
- Optional / disableable gamification

---

### H. Growth System (Tree / Pet)

- Virtual growth entity
- Growth tied to daily progress
- Visual health & evolution stages
- Cosmetic customization

---

### I. Activity Heatmap

- GitHub-style contribution calendar
- Progress-based color intensity
- Project & time filters
- Daily detail inspection

---

### J. Integrations

- Google Drive sync
- GitHub activity import
- Strava activity import
- Manual data fallback
- Background sync jobs

---

### K. Insights & Observability

- Daily reflection notes
- Weekly & monthly summaries
- Progress trends & balance metrics
- Debug view (scoring & data sources)

---

## Tech Stack

- React
- NestJS
- Node.js
- MongoDB (optional)
