## Zenin

**Description**: Gamified productivity app that is very data driven and lets you earn maximum dopamine through real life achievements you set yourself to.
**Features**:
- Goal plan that lets GPT generate quests for you
- Dopamine driven reward system
- Group quests into "campaigns", and time periods into "arcs"
- Level up and earn stats based on tasks
- Make personality test to further fit you
- Reward shop to spend coins to unlock custom rewards
- Options to add cooldowns and cost scaling
- Set up "boss fights"
- Different abilities based on clases
- Calendar, streaks, consistency & mood
- Daily/weekly stats
- XP trends, productivity by category/time
- See what tasks you enjoy more
- GPT-suggested new goals based on past behavior
- Themes, emblems and custom emojis
- Notification system
- Sync with Google Calendar
- Voice input
**Stack**:React, NextJS, ShadCN, TailwindCSS, Framer Motion, Supabase, Electron, MongoDB, PWA, TypeScript
